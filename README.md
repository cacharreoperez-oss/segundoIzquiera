# 🚀 Mis Proyectos de Bash y Linux - Raúl

¡Bienvenidos a mi repositorio del curso! Aquí he recopilado toda mi evolución, desde mi primer "Hola Mundo" hasta mi proyecto final de soporte técnico.

---

## 📂 Contenido del Curso

### [Nivel 1] - 🖐️ Saludo Personalizado
*   **Descripción:** Mi primer script interactivo que da la bienvenida con ASCII Art.
*   **Aprendizajes:** Variables, `read` y `cat <<EOF`.

### [Nivel 2] - 🖥️ Diagnóstico de Sistema
*   **Descripción:** Un script de reconocimiento que muestra IP, RAM y disco con barras de carga.
*   **Aprendizajes:** Comandos de sistema como `df`, `free` e `ip`.

### [Nivel 3] - 🧠 Quiz Interactivo
*   **Descripción:** Un juego de preguntas y respuestas sobre Linux con sistema de rangos (Jedi/Maestro).
*   **Aprendizajes:** Arrays, bucles `case` y validación de sinónimos.

### [Nivel 4] - 🛰️ Ground Control (Redes)
*   **Descripción:** Arquitectura Cliente-Servidor. Envío de telemetría por red mediante HTTP POST.
*   **Aprendizajes:** `curl`, servidores Python y comunicación entre máquinas.

### [Nivel 5] - 🤖 Asistente de Soporte Familiar (Proyecto Final)
*   **Descripción:** Mi gran proyecto. Una interfaz de **estética Neón** para ayudar a mi familia a arreglar el PC (Internet, Audio, RAM) con un "Botón del Pánico".
*   **Aprendizajes:** Diseño de interfaz (UX), automatización de tareas y gestión de errores.

---

## 🛠️ Cómo ejecutar el proyecto final
1. Entra en la carpeta `Nivel5_AsistenteFamiliar`.
2. Ejecuta `./asistente.sh`.
3. ¡Disfruta de la interfaz de neón! 🎨
